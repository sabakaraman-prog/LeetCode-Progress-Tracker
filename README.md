# LeetCode Tracker

## Overview
A React-based web application for tracking LeetCode progress, including solved problems, goals, and activity streaks. The app allows users to add and delete problems while maintaining persistent progress using localStorage.

## Features
- Add and delete solved problems
- Track total solved problems by difficulty (Easy, Medium, Hard)
- Dynamic progress tracking toward goals
- Streak calculation based on consecutive daily activity
- Persistent data using localStorage
- Responsive UI with reusable components

## Tech Stack
- React (Vite)
- JavaScript (ES6+)
- CSS
